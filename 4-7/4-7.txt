【main.js】
$(function(){
  $('.toggle_btn').on('click', function() {
    if ($('#header').hasClass('open')) {
      $('#header').removeClass('open');
    } else {
      $('#header').addClass('open');
    }
  });


  $('#mask').on('click', function() {
    $('#header').removeClass('open');
  });


  $('#navi a').on('click', function() {
    $('#header').removeClass('open');
  });


  $('a[href^="#"]').click(function(){
    let href= $(this).attr("href");
    let target = $(href == "#" || href == "" ? 'html' : href);
    let position = target.offset().top;
    $("html, body").animate({scrollTop:position}, 600, "swing");
    return false;
  });




  $('.slick-area').slick({
    arrows: false,
    centerMode: true,
    centerPadding: '100px',
    slidesToShow: 3,
    responsive: [
      {
        breakpoint: 768,
        settings: {
          centerPadding: '50px',
          slidesToShow: 1
        }
      }
    ]
  });


  $(window).scroll(function() {
    $('.fadein').each(function() {
      let scroll = $(window).scrollTop();
      let target = $(this).offset().top;
      let windowHeight = $(window).height();
      if (scroll > target - windowHeight + 200) {
        $(this).css('opacity','1');
        $(this).css('transform','translateY(0)');
      }
    });
  });
});

ブレークポイントで止まっていたプログラムの続きを実行する。
複数のブレークポイントを設定することで想定通りの処理順になっているか確認することができる。


②ステップオーバーとはどういった機能かお答えください
現在止まっている行で関数が実行されている場合に、その関数の内部の処理をすべて実行して次の行で止まる。


③ステップアウトとはどういった機能かお答えください
現在処理が止まっている関数内の残りの処理をすべて実行し、関数を呼び出している行へ移動する。


④ステップインとはどういった機能かお答えください
ブレークポイントで処理が止まっている時、その行のプログラムはまだ実行されていない状態。
ステップインを実行すると、現在の行のプログラムを実行し、次の処理の行へ移動してそこでまたプログラムが止まるので、プログラムが実行される処理の流れを1つひとつ確認できる。


⑤main.jsの27行目から41行目はどの動きをJavaScriptで実装しているか説明してください

・slidesToShow: 3  画面に表示するスライドの数を設定している。


⑥main.jsの27行目、$('.slick-area').slick({のみにブレイクポイントを置き、
ブレークポイントで止まっていたプログラムの続きを実行×2回、現在処理が止まっている関数内の残りの処理をすべて実行し、関数を呼び出している行へ移動したとき

・中央配置のパディングが100pxから50pxに変更される。
